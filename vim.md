# notes for vim shortcuts and workflows

## working with files

:w foo.txt	save As
## navigation

:tabedit filename
gt	go to next tab
gf	go to highlighted file
C+o	go back from file
analysis modules/

## vertical selection

C+v	enter vertical select mode
i	insert before cursor on all lines

## crud

o	insert new line below an enter insert mode
O	insert line above
dw	delete to end of word
d$	d
